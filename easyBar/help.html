Update:

 Allows you to change the current link to a new one if the they are in the same domain
(this comes in handy when you reading a book or watching a tv show)
just drag and drop the link you want to change to box.

Add tag:

 This function allow the user to add an identifier to each link incase if the user has more than on link from the same domain.
All you have to do is darg and drop the link you want to the box and give the identifier on popup box.

Delete:

 You can use this in two ways either you can drag and drop the link or you can press the button the delete links.
Warning: IF YOU PRESS DELETE IT Wll DELETE THE LAST LINK WITHOUT WARNING.
